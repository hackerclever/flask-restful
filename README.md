# flask-restful

# Flask RESTful API
- Get start for Flask RESTful API
- waitress for production

# Build image
- run this command to build image name 'flask-restful'
```
docker build -t flask-restful .
```

# Run containner with docker-compose
- run all service on docker-compose file
```
docker-compose up
```

# Run containner with docker-compose specific service
- run specific service flask-restful
```
docker-compose run flask-restful
```
